What is a scalar? A single number — magnitude only, no direction.
\( 22, \quad -3.7, \quad \pi \)
Geometrically: a point on a 1D number line.
In ML: a prediction output, a loss value, a learning rate — all scalars.
What is a vector in ML? An ordered list of scalars, also called a feature vector. Each entry is one measured property of a data point.
\( \vec{v} = \begin{bmatrix} 2100 \\ 3 \end{bmatrix} \leftarrow \text{house: 2100 sqft, 3 beds} \)
Geometrically: an arrow from the origin to a specific coordinate in space. The number of entries = the number of dimensions.
What is a matrix? A rectangular grid of numbers — a collection of vectors stacked as rows. It represents a complete dataset.
\( A = \begin{bmatrix} 2100 & 3 \\ 1500 & 2 \\ 3000 & 4 \end{bmatrix} \leftarrow 3 \text{ houses} \times 2 \text{ features} \)
Shape notation: m × n = m rows × n columns.
In a data matrix, what does each row represent? Each column?
  • Row = one data point (one observation / sample)
  • Column = one feature (one measured property across all samples)
\( X \in \mathbb{R}^{m \times n}: \quad m \text{ samples}, \quad n \text{ features} \)
A dataset of 1,000 houses with 5 features → X is 1000 × 5.
State the hierarchy: scalar → vector → matrix
  • Scalar: one value (0D)
  • Vector: a column of scalars — one data point (1D)
  • Matrix: a grid of vectors — an entire dataset (2D)
  • Tensor: higher-dimensional array (e.g. image = 3D)
Each level adds one more dimension of structure.
Why do we treat data geometrically? Every data point is a coordinate in space. Geometry gives us powerful tools:
  • Distance = similarity between data points
  • Direction = relationship between features
  • Intersection = the point satisfying all constraints
A house has 2100 sq ft and 3 bedrooms. Write its feature vector.
\( \vec{v} = \begin{bmatrix} 2100 \\ 3 \end{bmatrix} \)
This vector is a point in 2D space where the x-axis = square footage and the y-axis = bedrooms. Every house in the dataset is its own arrow pointing to its own location in this space.
Geometrically, what is a vector? An arrow from the origin (0, 0, …) to a specific coordinate.
  • Arrow's length = magnitude of the vector
  • Arrow's direction = the combination of its components
Two vectors are equal if and only if they have the same magnitude AND direction — regardless of where in space they're drawn.
Write the general form of a linear equation in ML notation.
\( y = w_1 x_1 + w_2 x_2 + \cdots + w_n x_n + b \)
  • \(y\) — label / prediction (output)
  • \(x_i\) — features (inputs you know)
  • \(w_i\) — weights (importance of each feature)
  • \(b\) — bias (y-intercept)
What do weights \(w_i\) represent in a linear model? Weights measure the importance / contribution of each feature.
  • Large \(|w_i|\) → feature \(x_i\) strongly drives the output
  • \(w_i = 0\) → feature has no effect
  • Negative \(w_i\) → feature is inversely related to the output
Training a model = finding the right weights. The learning algorithm adjusts them until predictions match the data.
What does the bias \(b\) do in a linear model? The bias is the y-intercept. It shifts the model so it doesn't have to pass through the origin.
\( y = wx + b \quad \Rightarrow \quad b = y \text{ when } x = 0 \)
Without bias, the line is forced through (0, 0) — unrealistic for most real data (a house with 0 sqft should not predict a $0 price).
Why is \(y = w_1 x_1 + w_2 x_2 + b\) called linear? It uses only addition and scalar multiplication — no squaring, no exponents, no products of variables.
  • In 2D: produces a straight line
  • In 3D: produces a flat plane
  • In nD: produces a hyperplane
What does training a model mean in terms of linear equations? Training means finding the weights \(w_1, w_2, \ldots, b\) that make the model's predictions best match the actual data.
Each data point gives one equation. The full dataset is a system of equations. Training = solving that system for the unknown weights.
What does a single linear equation look like geometrically in 2D? A straight line.
\( y = mx + b \)
Every point on the line satisfies the equation.
In ML: the regression line of a linear model, or the decision boundary of a linear classifier.
What does a single linear equation look like geometrically in 3D? A flat plane.
\( w_1 x_1 + w_2 x_2 + w_3 x_3 = b \)
In 3D, one equation constrains the solution to lie on an infinite flat sheet. You need three independent equations to pinpoint a single point.
How does \(y = mx + b\) extend to ML with many features? The slope \(m\) becomes multiple weights, one per feature:
\( y = w_1 x_1 + w_2 x_2 + \cdots + w_n x_n + b \)
In vector notation:
\( y = \vec{w}^{\,T} \vec{x} + b \)
Still a linear combination — just in a higher-dimensional space.
What is a system of linear equations? A collection of linear equations that must all be satisfied simultaneously. Each equation is one constraint.
\( \begin{cases} x + y = 5 \\ x - y = 1 \end{cases} \)
Solving the system = finding values that make every equation true at once.
What does solving a system mean geometrically? Finding the point of intersection — the one location in space that lies on every line (2D) or plane (3D) simultaneously.
If the lines/planes don't meet at exactly one point, the system has no solution or infinitely many solutions.
Name the 3 possible outcomes when solving a system.
  • Unique solution — lines/planes meet at exactly one point
  • No solution — lines/planes are parallel (never meet)
  • Infinite solutions — lines/planes are identical (overlap entirely)
Only a unique solution means the data has fully determined the model.
What causes a system to have no solution? Two equations have the same left-hand side but different right-hand values.
\( x + y = 10 \quad \text{and} \quad x + y = 20 \)
Geometrically: parallel lines — same slope, different intercepts. No pair of numbers can sum to both 10 and 20 simultaneously.
What causes a system to have infinitely many solutions? One equation is a scaled copy of another — they describe the same geometric object.
\( x + y = 3 \quad \text{and} \quad 2x + 2y = 6 \)
Geometrically: identical lines lying on top of each other. Any point on the line is a valid solution.
What is an augmented matrix? A compact encoding of a system: coefficients on the left of a bar, constants on the right.
\( 2x + y = 5, \; x - y = 1 \;\Rightarrow\; \left[\begin{array}{cc|c} 2 & 1 & 5 \\ 1 & -1 & 1 \end{array}\right] \)
Each row = one equation. Each column = one variable. This compact form is what Gaussian Elimination operates on.
In ML, what does each equation in a system represent? Each equation represents one data point imposing a constraint on the model weights.
100 data points with 2 unknowns (w, b) = 100 equations in 2 unknowns. Solving = finding the weights consistent with all observations.
What does a unique solution mean for an ML model? The data has fully determined the model — one set of weights fits all observations.
"We found the rule."

Infinite solutions → "Can't tell which rule."
No solution → "No such rule exists."
What does linearly independent mean for equations/vectors? Each equation/vector points in a genuinely new direction that cannot be produced by scaling or combining the others.
No equation = linear combination of the rest
Each adds real new information, narrowing the solution further toward a unique point.
What does linearly dependent mean? At least one equation/vector can be expressed as a linear combination of the others.
e.g., Row 3 = 2 × Row 1 + Row 2
It adds zero new information. The system collapses — it can no longer pinpoint a unique solution.
What is the rank of a matrix? Rank = the number of linearly independent rows — the true dimensionality of the data.
  • Rank 3 in a 3×3 system → 3 independent equations → unique solution possible
  • Rank 2 in a 3×3 system → data lives on a 2D plane
  • Rank 1 → everything collapses to a single line
What does full rank mean? Rank equals the smaller dimension of the matrix — all rows/columns are linearly independent.
Full rank n×n matrix: rank = n
A full-rank square matrix is non-singular: unique solution, non-zero determinant, invertible.
What is a singular matrix? A square matrix that is not full rank — at least one row is linearly dependent on the others.
  • \(\det(A) = 0\)
  • No unique solution
  • No inverse exists
A singular data matrix means the model cannot be uniquely determined from the data.
What are the two failure modes caused by linear dependence? 1. Infinite solutions (redundant equations):
One equation is a scaled copy → lines/planes overlap → infinitely many intersections.

2. No solution (contradictory equations):
Same direction, different targets → parallel lines/planes → no intersection.
Explain the "shadow analogy" for rank. A 3D object casts a shadow. Depending on the light angle:
  • 2D shadow (Rank 2): depth lost
  • 1D shadow — a line (Rank 1): two dimensions lost
Rank = the number of genuine dimensions that survived.
Why are redundant features dangerous in ML? If one feature is a linear combination of others, the data matrix becomes rank-deficient → singular.
  • \(\det(X^T X) = 0\)
  • \((X^T X)^{-1}\) doesn't exist
  • Linear regression formula breaks
PCA is fundamentally about finding and removing redundant directions before they cause singularity.
Give a real-world example of linear dependence in ML features. Including both Temperature in °C and Temperature in °F:
\( F = 1.8C + 32 \)
One is always derivable from the other — same geometric direction, zero new information. Including both makes the data matrix rank-deficient.
Fill in: Rank = n (full) → ?    Rank < n → ?
  • Rank = n: All independent · Unique solution · det ≠ 0 · Inverse exists · Non-singular
  • Rank < n: At least one dependent equation · No unique solution · det = 0 · No inverse · Singular
What does the determinant measure geometrically? How much a matrix transformation stretches or squashes space — the signed area/volume scaling factor.
  • \(\det = 6\): unit square → parallelogram with area 6
  • \(\det = -6\): same area, orientation flipped (mirror)
  • \(\det = 0\): space squashed flat — a dimension was destroyed
Give the formula for the 2×2 determinant.
\( \det\!\begin{pmatrix} a & b \\ c & d \end{pmatrix} = ad - bc \)
Multiply the main diagonal \((a \times d)\), subtract the off-diagonal \((b \times c)\).
Example: \(\det\!\begin{pmatrix}3&1\\1&2\end{pmatrix} = 6 - 1 = 5 \neq 0\) → non-singular.
What does \(\det(A) = 0\) tell you? The transformation permanently destroyed a dimension — squashed 3D space into a flat plane (or a plane into a line).
  • Matrix is singular
  • Rank is deficient
  • No unique solution to \(Ax = b\)
  • No inverse exists
What does \(\det(A) \neq 0\) guarantee?
  • Matrix is non-singular
  • Full rank — all equations are independent
  • A unique solution exists for \(Ax = b\)
  • The inverse \(A^{-1}\) exists
\(\det \neq 0\) is the single-number green light: the system is fully solvable.
What does a negative determinant mean? The transformation flipped orientation — like a mirror reflection.
det = −6: area/volume is still 6, handedness reversed.
The sign encodes orientation. The magnitude \(|\det|\) gives the area/volume scaling factor. Negative det is still perfectly invertible.
State the Rule of Sarrus for a 3×3 determinant.
\( \det(A) = (aei + bfg + cdh) - (ceg + afh + bdi) \)
  • Add the three "down-right" diagonal products
  • Subtract the three "down-left" diagonal products
Result = signed volume of the parallelepiped formed by the three column vectors.
List the four equivalent conditions for a singular matrix. All four are the same underlying event:
  • \(\det(A) = 0\)
  • Rank-deficient (rank < n)
  • No unique solution to \(Ax = b\)
  • No inverse \(A^{-1}\)
What does \(\det(A) = 8\) tell you about the transformation? The transformation stretches space by a factor of 8.
  • Unit square (area 1) → parallelogram with area 8
  • Unit cube (volume 1) → parallelepiped with volume 8
Positive sign → orientation preserved. Non-zero → matrix is non-singular.
What is the goal of Gaussian Elimination? Transform the augmented matrix step-by-step into Reduced Row Echelon Form (RREF), where the coefficient side becomes the Identity Matrix and the solution is directly readable on the right.
\( \left[\begin{array}{cc|c}1&0&x_1\\0&1&x_2\end{array}\right] \) — done!
What is the Identity Matrix? A square matrix with 1s on the main diagonal and 0s everywhere else.
\( I_2 = \begin{bmatrix}1&0\\0&1\end{bmatrix}, \quad I_3 = \begin{bmatrix}1&0&0\\0&1&0\\0&0&1\end{bmatrix} \)
The "number 1 of the matrix world": \(AI = IA = A\). It performs no transformation — every vector is left unchanged.
What is Reduced Row Echelon Form (RREF)? A matrix where:
  • Each pivot = 1
  • All entries above and below each pivot are 0
  • Coefficient side = Identity Matrix
\( \left[\begin{array}{cc|c}1&0&1\\0&1&2\end{array}\right] \Rightarrow x=1, \; y=2 \)
Solution is directly visible — no further calculation needed.
What is the difference between REF and RREF? REF (Row Echelon Form): Zeros below each pivot only (staircase). Pivots need not be 1. Requires back-substitution.

RREF (Reduced REF): Each pivot = 1, zeros above AND below. Solution directly visible on the right.
RREF is the fully simplified final state. REF is a useful intermediate step.
List the 3 elementary row operations.
  • Swap: Exchange any two rows   \(R_1 \leftrightarrow R_2\)
  • Scale: Multiply a row by any non-zero number   \(R_1 \leftarrow kR_1\)
  • Combine: Add a multiple of one row to another   \(R_2 \leftarrow R_2 - 2R_1\)
Why are elementary row operations safe? Each operation changes the appearance of the equations but never the solution — the intersection point is invariant.
Like translating a contract: the words change, the agreement doesn't.
What is a pivot in Gaussian Elimination? The first non-zero entry in a row (after earlier steps). At each step, the pivot zeroes out all other entries in its column — above and below.
\( \left[\begin{array}{cc|c}\mathbf{1}&1&3\\2&4&10\end{array}\right] \) — bold 1 is the pivot
Pivots march down the diagonal, driving the matrix toward the identity pattern.
Solve \(x + y = 3\) and \(2x + 4y = 10\) using Gaussian Elimination. Step 1 — Augmented matrix:
\( \left[\begin{array}{cc|c}1&1&3\\2&4&10\end{array}\right] \)
Step 2 — \(R_2 \leftarrow R_2 - 2R_1\):
\( \left[\begin{array}{cc|c}1&1&3\\0&2&4\end{array}\right] \)
Step 3 — \(R_2 \leftarrow \frac{1}{2}R_2\), then \(R_1 \leftarrow R_1 - R_2\):
\( \left[\begin{array}{cc|c}1&0&1\\0&1&2\end{array}\right] \)
Answer: \(x = 1, \; y = 2\)
During elimination, how do you spot no solution vs infinite solutions? No solution: A row becomes \([0 \;\; 0 \;|\; c]\) with \(c \neq 0\) — an impossible constraint (\(0 = c\)).

Infinite solutions: A row becomes \([0 \;\; 0 \;|\; 0]\) — the equation vanishes entirely, leaving a free variable unconstrained.
What is the inverse matrix \(A^{-1}\)? The matrix that perfectly reverses whatever \(A\) does.
\( A^{-1}A = AA^{-1} = I \)
If \(A\) maps \(\vec{x} \mapsto \vec{b}\) (i.e. \(Ax = b\)), then \(A^{-1}\) maps back: \(\vec{b} \mapsto \vec{x}\).
How do you solve \(Ax = b\) using the inverse? Multiply both sides on the left by \(A^{-1}\):
\( Ax = b \;\Rightarrow\; A^{-1}Ax = A^{-1}b \;\Rightarrow\; x = A^{-1}b \)
In practice, prefer np.linalg.solve(A, b) — it's faster and more numerically stable than computing \(A^{-1}\) explicitly.
When does the inverse \(A^{-1}\) exist? If and only if \(\det(A) \neq 0\) — equivalently, when \(A\) is:
  • Non-singular
  • Full rank
  • Has a unique solution for every right-hand side \(b\)
These are all the same condition.
Why can't a singular matrix be inverted? Because the transformation permanently destroyed information.
If \(A\) squashes 3D space into a flat 2D plane, multiple 3D inputs map to the same output — you can't reconstruct the unique original. Inversion requires a lossless transformation.
Explain the "egg analogy" for matrix inversion. Think of matrix \(A\) as scrambling an egg.
  • \(A^{-1}\) would unscramble it — only possible if no information was lost
  • A singular matrix is like cooking the egg — irreversible
The inverse exists only when the original transformation was lossless and reversible.
Write the closed-form linear regression formula using the matrix inverse.
\( \hat{w} = (X^T X)^{-1} X^T y \)
  • \(X\): feature matrix (m × n)
  • \(y\): label vector
  • \(\hat{w}\): optimal weight vector
This requires \(X^T X\) to be invertible — i.e., no redundant features.
What happens if \(X^T X\) is singular in linear regression? The formula \(\hat{w} = (X^T X)^{-1} X^T y\) breaks down completely — the inverse doesn't exist.
Correlated / redundant features cause this. The model has infinitely many equally valid weight vectors. Fix: remove redundant features, add regularisation (Ridge), or use the pseudoinverse.
Why prefer np.linalg.solve(A,b) over np.linalg.inv(A) @ b? np.linalg.solve uses a numerically stable Gaussian Elimination internally — never computes the full inverse.
  • Faster — avoids extra matrix operations
  • More accurate — less floating-point rounding error on large systems
Computing \(A^{-1}\) explicitly is rarely necessary in practice.
Solve \(Ax = b\) in NumPy. x = np.linalg.solve(A, b)

Full pattern:
A = np.array([[1,2],[3,4]])
b = np.array([5, 11])
x = np.linalg.solve(A, b)
Uses stable Gaussian Elimination. Preferred over inv(A) @ b.
Compute the determinant in NumPy. det = np.linalg.det(A)

Check singularity safely:
if np.isclose(det, 0):
    print("Singular!")
Use np.isclose not det == 0 — floating-point rarely gives exact zero.
Find the rank of a matrix in NumPy. rank = np.linalg.matrix_rank(A)

Interpret result:
n = A.shape[0]
full_rank = (rank == n)
If rank < n → matrix is singular → no unique solution → no inverse.
Compute the matrix inverse in NumPy. A_inv = np.linalg.inv(A)

Always check first:
if not np.isclose(np.linalg.det(A), 0):
    A_inv = np.linalg.inv(A)
Calling inv on a singular matrix raises a LinAlgError.
Why use np.isclose(det, 0) instead of det == 0? Floating-point arithmetic rarely produces an exact zero even when the true answer is 0.
Example: a near-singular matrix might give det = 1.2e-16 instead of 0. det == 0 misses it; np.isclose(det, 0) correctly catches it.
\( \text{isclose}(a,b) : \; |a - b| \leq \text{atol} + \text{rtol} \cdot |b| \)
What does the @ operator do in NumPy? @ is the matrix multiplication operator.
A @ b  ≡  np.matmul(A, b)
X.T @ X — computes \(X^T X\)
A_inv @ b — applies inverse to b
Do not confuse with * which is element-wise multiplication.
Write the complete NumPy workflow: represent → check → solve. import numpy as np
A = np.array([[1,2],[3,4]])
b = np.array([5, 11])

det = np.linalg.det(A)
rank = np.linalg.matrix_rank(A)

if not np.isclose(det, 0):
  x = np.linalg.solve(A, b)
  print(x)  # [1. 2.]
What do rank, determinant, and inverse all measure? They are three different views of the same underlying event: whether the matrix transformation lost a dimension.
  • Geometrically: Did the transformation squash space flat? (det = 0)
  • Information-wise: Are any equations redundant? (rank < n)
  • Algebraically: Can we reverse the transformation? (no inverse)

In one sentence: why does ML need linear algebra? Because data is geometric, predictions are linear combinations, and training is solving a system of equations for unknown weights — all requiring the language and tools of linear algebra.
Every forward pass, gradient update, and weight matrix in a neural network is a matrix operation.
Map the three system outcomes to their ML meanings.
  • Unique solution → "We found the rule." — data fully determines the model
  • Infinite solutions → "Can't tell which rule." — underdetermined, redundant data
  • No solution → "No such rule exists." — contradictory data, linear model can't fit
What does Gaussian Elimination actually compute? GE computes the inverse implicitly — it finds exactly what \(A^{-1}\) does to \(b\) without ever forming \(A^{-1}\).
\( [A \;|\; b] \;\xrightarrow{\text{GE}}\; [I \;|\; A^{-1}b] = [I \;|\; x] \)
To compute \(A^{-1}\) directly: run GE on \([A \;|\; I]\) to get \([I \;|\; A^{-1}]\).
If \(\det(A) = 0\), list everything you know about A.
  • Matrix is singular
  • Rank is deficient (rank < n)
  • At least one equation is linearly dependent
  • No unique solution to \(Ax = b\)
  • No inverse \(A^{-1}\)
  • Transformation collapsed a dimension to zero volume
  • In ML: model is underdetermined or contradictory
How does linear dependence connect to PCA? PCA finds and discards redundant directions — near-zero-variance dimensions that are effectively linearly dependent.
PCA rotates to the truly independent axes, directly addressing rank deficiency by working only in the subspace of genuine information.
Why is \(y = \vec{w}^{\,T} \vec{x} + b\) a scalar output from vector inputs? Because \(\vec{w}^T \vec{x}\) is the dot product — multiply corresponding components and sum:
\( \vec{w}^T \vec{x} = w_1 x_1 + w_2 x_2 + \cdots + w_n x_n \in \mathbb{R} \)
A row vector times a column vector always collapses to a single number — the prediction \(y\).
Distinguish: dimension of the space vs rank of the data matrix.
  • Dimension of space = number of features (columns of X) — the apparent dimensionality
  • Rank = the true dimensionality — how many genuinely independent directions the data spans
A 1,000-feature dataset might only have rank 50 if features are correlated. PCA reveals this intrinsic dimension.
Geometrically, what does \(A\vec{x}\) do to a vector? It applies a linear transformation — rotating, scaling, shearing, or reflecting the vector in space.
\( A\vec{x} = \vec{b} \) maps point \(\vec{x}\) to new point \(\vec{b}\)
The columns of \(A\) tell you where the standard basis vectors land. Every other vector follows by linearity.
In a neural network, each layer is a matrix multiplication — a geometric transformation of the data cloud.
