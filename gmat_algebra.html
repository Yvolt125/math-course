Lesson 1

Linear Equations

Learning Objectives
  • Solve single-variable linear equations by isolating the variable.
  • Solve systems of two linear equations using substitution and elimination.
  • Recognise when a system has no solution, infinite solutions, or a unique solution.
📌 Core Concept

A linear equation has one or more variables each raised to the first power. Solving means isolating the variable on one side. Every valid operation performed on one side must be performed on the other to maintain equality.

How to Approach It
  1. 1Distribute and combine like terms on each side of the equation first.
  2. 2Move all variable terms to one side and all constant terms to the other.
  3. 3Divide both sides by the coefficient of the variable to isolate it.
  4. 4Verify your answer by substituting it back into the original equation.
Example

If 3x − 7 = 2x + 5, what is the value of 4x?

(A) 12(B) 24(C) 36(D) 48(E) 52
▶ Show Answer

Answer: (D). 3x − 7 = 2x + 5 → x = 12. The question asks for 4x, not x — so 4 × 12 = 48. Common trap: selecting (A) 12 after solving for x and forgetting to compute 4x.

Common Traps
  • In systems of equations, multiplying both sides of an equation by a variable without checking whether that variable could be zero — this can create extraneous solutions.
  • Solving for x when the question asks for an expression like 2x + 1 — compute the full expression, not just x.
Key Takeaways
  • For elimination: multiply equations to match coefficients, then add or subtract to eliminate one variable.
  • If two equations are multiples of each other, there are infinite solutions. If they are parallel (same slope, different intercept), there is no solution.
  • Always verify the solution satisfies both original equations.
💡 Quick Tip

For two equations with two unknowns, use substitution if one equation is already solved for a variable. Otherwise use elimination — multiply one or both equations so a variable cancels when you add them.

Chapter 2

Inequalities

📌 Core Concept

Inequalities work like equations except the direction of the inequality sign flips when you multiply or divide both sides by a negative number. This single rule is the source of most inequality errors on the GMAT.

How to Approach It
  1. 1Solve like an equation: isolate the variable using addition, subtraction, multiplication, and division.
  2. 2Flip the inequality sign whenever you multiply or divide both sides by a negative number.
  3. 3Express the solution as a range (e.g., x > 3) and verify by checking a value in that range.
  4. 4For compound inequalities, perform operations on all three parts simultaneously.
Example

If −2x + 3 > 7, which of the following must be true?

(A) x > 2(B) x > −2(C) x < −2(D) x < 2(E) x = −2
▶ Show Answer

Answer: (C). −2x + 3 > 7 → −2x > 4 → x < −2 (flip the sign when dividing by −2). Common trap: forgetting to flip the inequality sign when dividing by a negative number, giving x > −2.

💡 Quick Tip

When the variable could be positive or negative — for example, in absolute value problems — split into two cases: one where the expression inside is positive, and one where it is negative.

Chapter 3

Quadratics

📌 Core Concept

A quadratic equation has the form ax² + bx + c = 0. Solutions are found by factoring, completing the square, or the quadratic formula. Factoring is always the fastest approach — try it first before reaching for the formula.

How to Approach It
  1. 1Rearrange all terms to one side so the equation is in standard form: ax² + bx + c = 0.
  2. 2Try factoring first: find two numbers that multiply to ac and add to b, then split the middle term.
  3. 3If factoring fails, use the quadratic formula: x = (−b ± √(b²−4ac)) / 2a.
  4. 4Set each factor equal to zero and solve for both roots.
Example

What are the solutions to x² − 5x + 6 = 0?

(A) x = 1 and x = 6(B) x = 2 and x = 3(C) x = −2 and x = −3(D) x = −1 and x = 6(E) x = 2 and x = −3
▶ Show Answer

Answer: (B). Find two numbers that multiply to 6 and add to −5: that is −2 and −3. So (x − 2)(x − 3) = 0, giving x = 2 and x = 3. Common trap: choosing (C) because −2 × −3 = 6, but −2 + −3 = −5 ✓ — both signs must be checked together.

💡 Quick Tip

The discriminant b²−4ac tells you the number of real solutions before you solve: greater than 0 means two solutions, equal to 0 means one, and less than 0 means none (no real roots).

Chapter 4

Functions

📌 Core Concept

A function f(x) takes an input x and returns a unique output. On the GMAT, function questions often use unusual notation like f★g or ask you to evaluate composite functions. The key is always treating the notation as a pure substitution rule.

How to Approach It
  1. 1Substitute the given input directly into the function rule, replacing every instance of the variable.
  2. 2For composite functions f(g(x)), evaluate the inner function g(x) first, then pass the result to f.
  3. 3For "which value satisfies f(x) = k" questions, set the expression equal to k and solve algebraically.
  4. 4For custom-symbol functions (e.g., a★b = 2a − b), apply the definition exactly as given.
Example

If f(x) = 3x² − 2, what is the value of f(f(1))?

(A) 1(B) 3(C) 7(D) 25(E) 73
▶ Show Answer

Answer: (A). First evaluate the inner function: f(1) = 3(1)² − 2 = 3 − 2 = 1. Then f(f(1)) = f(1) = 1. Common trap: computing f(1) = 1 and then attempting f(1) a second time without realising the output feeds back in — here it loops to the same value.

💡 Quick Tip

Treat function notation as a substitution machine — f(3) simply means replace every x in the rule with 3. Do not overthink the notation; the mechanics are always the same.
