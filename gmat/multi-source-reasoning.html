Lesson 1

Navigating Tabs of Information

Objectives
  • Efficiently locate specific information across multiple information tabs.
  • Identify which tab is relevant to each question before searching.
  • Avoid importing assumptions from one tab into another — each tab may present a different perspective or data set.
📌 Core Concept

Multi-Source Reasoning presents 2–3 tabs of information (text passages, tables, or charts) and asks 3 questions that may require synthesising data across tabs. The challenge is information management — knowing which tab contains the relevant data and reading each source on its own terms.

MSR is effectively open-book: you can return to any tab at any time. The skill is knowing which tab to return to, not memorising every detail upfront.

How to Approach It
  1. 1Skim all tabs quickly for structure — what type of info is in each tab (text, table, chart)?
  2. 2For each question, identify the relevant tab(s) first before going back to re-read them.
  3. 3Locate and read the specific section carefully.
  4. 4Answer based solely on what the tabs state — do not import outside knowledge.
Example

Tab 1 — Memo: "The new pricing policy, effective Q3, sets a 15% discount on all orders above 100 units. Orders of 100 units or fewer are billed at the standard list price of $50 per unit."

Tab 2 — Orders Table:

OrderUnitsQuarter
A80Q3
B150Q3
C100Q2

According to the information in both tabs, which statement correctly identifies the tab needed to determine the price paid for Order B?

(A) Tab 1 only — the memo gives the applicable price(B) Tab 2 only — the table gives the applicable price(C) Both tabs are needed to determine the correct price(D) Neither tab — the price cannot be determined
▶ Show Answer

Answer: (C). Tab 2 tells us Order B has 150 units in Q3. Tab 1 (the memo) tells us the discount policy: orders above 100 units get a 15% discount. Both tabs are needed — Tab 2 provides the quantity and quarter, Tab 1 provides the pricing rule that determines whether the discount applies. Common trap: choosing (A) because the memo has the price — but you need Tab 2 to confirm the order details first.

Common Traps
  • Importing general knowledge instead of using only the given tab information — MSR is a closed-information question type.
  • Assuming information from Tab 1 applies to Tab 2 when the tabs present different scenarios or perspectives.
Summary
  • Skim all tabs for structure before answering — establish what type of information is in each.
  • Each question maps to specific tab(s) — identify them first, then re-read only what's needed.
  • Use only the given information; outside knowledge is not tested here.
💡 Quick Tip

Label each tab in your head: "Tab 1 = financials," "Tab 2 = memo," "Tab 3 = chart." This mental map saves navigation time under pressure and prevents re-reading the wrong tab.

Lesson 2

Cross-Referencing & Verbal-Quant Integration

Objectives
  • Synthesise information from two or more tabs to answer a single question.
  • Apply quantitative reasoning to data presented in a text or email format.
  • Distinguish between what the tabs state directly and what can be inferred by combining them.
📌 Core Concept

The hardest MSR questions require combining data from multiple tabs — for example, a table in Tab 1 and an interpretive email in Tab 2. The key distinction is between information explicitly stated, information that can be logically inferred by combining tabs, and information that is neither stated nor inferable.

MSR sets typically mix a text passage with a data table. The text tab provides context, definitions, and qualifications; the data tab provides the specific numerical values. Always apply verbal qualifications before computing.

How to Approach It
  1. 1Identify all tabs whose information is needed for the question.
  2. 2Extract the specific data points from each tab independently.
  3. 3Combine them (arithmetically or logically) to produce the answer.
  4. 4Ask: "Is this conclusion directly stated, inferable, or neither?" — only the first two are valid answers.
Example

Tab 1 — Memo (same as Lesson 1): Standard list price = $50/unit. Orders above 100 units in Q3+ receive a 15% discount.

Tab 2 — Orders Table (same as Lesson 1): Order A = 80 units Q3; Order B = 150 units Q3; Order C = 100 units Q2.

What was the total amount billed for Order B?

(A) $5,000(B) $6,375(C) $7,500(D) $8,500(E) Cannot be determined
▶ Show Answer

Answer: (B). From Tab 2: Order B = 150 units, Q3. From Tab 1: 150 > 100 and it is Q3, so the 15% discount applies. Price per unit = $50 × (1 − 0.15) = $50 × 0.85 = $42.50. Total = 150 × $42.50 = $6,375. Common trap: choosing (C) $7,500 by ignoring the discount, or (E) "cannot be determined" by failing to combine Tab 1 and Tab 2.

Common Traps
  • Choosing "cannot be determined" when the answer is actually inferable by combining the tabs — cross-tab inferences are often missed.
  • Over-inferring: drawing a conclusion that seems logical but is not actually supported by the tab text — inferences must be directly supported.
Summary
  • Cross-tab questions require explicit combination of data from 2+ tabs — find the connecting variable linking the two sources.
  • Inferences must be directly supported — not just plausible.
  • "Cannot be determined" is correct only when the tabs truly do not provide enough information, even in combination.
💡 Quick Tip

For inference questions, ask: "Does the tab explicitly say this, or can I derive it from what the tabs give me?" If neither, choose "cannot be determined." If either, the answer is supported.
