Chapter 1
Matplotlib Architecture

Matplotlib has two interfaces. The pyplot interface (plt.plot()) is quick and good for exploration. The object-oriented interface (fig, ax = plt.subplots()) gives you full control and is what you use in production code.

Figure and Axes — the core objects

Figure (the whole canvas)
Axes (one plot area)
Title · X-axis · Y-axis · Legend · Ticks · Labels
A Figure can contain multiple Axes (subplots).
import matplotlib.pyplot as plt
import numpy as np

# Object-oriented interface (recommended)
fig, ax = plt.subplots(figsize=(10, 5))  # width=10", height=5"

x = [1, 2, 3, 4, 5]
y = [12400, 15800, 9700, 18300, 14200]

ax.plot(x, y)
ax.set_title('Monthly Sales')
ax.set_xlabel('Month')
ax.set_ylabel('Sales ($)')
ax.grid(True, alpha=0.3)

plt.tight_layout()
plt.savefig('chart.png', dpi=150, bbox_inches='tight')
plt.show()

pyplot shortcut (quick exploration)

plt.plot(x, y, marker='o', color='steelblue')
plt.title('Sales')
plt.xlabel('Month')
plt.ylabel('Sales ($)')
plt.show()
Always use the OO interface in scripts and notebooks When building charts inside functions or loops, fig, ax = plt.subplots() prevents charts from bleeding into each other. The plt.xxx() shortcut always operates on the "current" Axes — which can be surprising in complex layouts.

Scatter Chart — relationships between two variables

# Hours studied vs exam score
hours  = [2, 3, 3.5, 4, 4.5, 5, 5.5, 6, 6.5, 7, 8, 9]
scores = [42, 51, 55, 63, 70, 72, 75, 80, 83, 85, 91, 94]
sizes  = [40]*12   # bubble size; can vary to add 3rd dimension

fig, ax = plt.subplots(figsize=(8, 5))

# Trend line
m, b = np.polyfit(hours, scores, 1)

ax.set_title('Study Hours vs Exam Score', fontsize=13, fontweight='bold')
ax.set_xlabel('Hours studied')
ax.set_ylabel('Score')
ax.legend()

Unequal Subplot Sizes — GridSpec

from matplotlib.gridspec import GridSpec

fig = plt.figure(figsize=(12, 6))
gs  = GridSpec(2, 3, figure=fig, hspace=0.4, wspace=0.35)

ax_main  = fig.add_subplot(gs[:, :2])    # spans all rows, left 2 cols
ax_top   = fig.add_subplot(gs[0, 2])     # top-right
ax_bot   = fig.add_subplot(gs[1, 2])     # bottom-right

ax_main.plot(dates, revenue)
ax_main.set_title('Revenue Trend (main)')

ax_top.bar(regions, q1_sales)
ax_top.set_title('Q1 by Region')

ax_bot.bar(regions, q2_sales, color='#27ae60')
ax_bot.set_title('Q2 by Region')
plt.tight_layout() vs fig.tight_layout() Both do the same thing — automatically adjust subplot spacing so labels don't overlap. Call it just before plt.savefig() or plt.show().
Chapter 6
Seaborn — Statistical Plots
When to reach for Seaborn first Open Seaborn instead of Matplotlib when you have a DataFrame and want a statistical view quickly:
  • Distribution of a column — sns.histplot(df, x='sales')
  • Compare distributions across groups — add hue='region' to almost any Seaborn call
  • Correlation matrix — sns.heatmap(df.corr(), annot=True)
  • Relationship between two numeric columns — sns.regplot() adds a regression line automatically
  • Spot outliers across groups — sns.boxplot() or sns.violinplot()
Switch to Matplotlib when you need precise control: exact positioning, custom annotations, complex multi-panel layouts, or pixel-perfect output for a report. Use both together with sns.xxx(ax=ax).

Seaborn is built on top of Matplotlib and is designed for statistical visualisation. It understands Pandas DataFrames natively — you just pass column names.

import seaborn as sns

sns.set_theme(style='whitegrid', palette='muted')  # global style

Distribution Plots

df = pd.read_csv('orders.csv')

# Histogram with KDE (kernel density estimate)
fig, ax = plt.subplots(figsize=(8, 4))
sns.histplot(df['order_value'], kde=True, bins=40, color='#5dade2', ax=ax)
ax.set_title('Order Value Distribution')

# Separate distributions per group
sns.histplot(data=df, x='order_value', hue='region',
             kde=True, alpha=0.5)

# KDE only (smooth density curve)
sns.kdeplot(data=df, x='order_value', hue='region', fill=True, alpha=0.3)

Box & Violin Plots — spread and outliers

fig, axes = plt.subplots(1, 2, figsize=(12, 5))

sns.boxplot(data=df, x='region', y='sales', palette='Set2', ax=axes[0])
axes[0].set_title('Sales Distribution by Region (Box)')

sns.violinplot(data=df, x='region', y='sales', palette='Set2',
               inner='quartile', ax=axes[1])
axes[1].set_title('Sales Distribution by Region (Violin)')

Chapter 7
Heatmaps & Pairplots

Heatmap — correlation or pivot table

import pandas as pd, seaborn as sns, matplotlib.pyplot as plt

# Correlation heatmap
numeric_cols = df.select_dtypes(include='number')
corr = numeric_cols.corr()

fig, ax = plt.subplots(figsize=(8, 6))
sns.heatmap(
    corr,
    annot=True,           # show numbers in cells
    fmt='.2f',            # 2 decimal places
    cmap='RdYlGn',        # diverging colormap
    center=0,             # 0 is neutral colour
    vmin=-1, vmax=1,
    linewidths=0.5,
    ax=ax
)
ax.set_title('Feature Correlation Matrix', pad=14)
plt.tight_layout()
# Pivot table heatmap — e.g. sales by region × month
pivot = df.pivot_table(values='sales', index='region', columns='month', aggfunc='sum')

fig, ax = plt.subplots(figsize=(10, 4))
sns.heatmap(pivot, annot=True, fmt=',d', cmap='YlOrRd', linewidths=0.5, ax=ax)
ax.set_title('Sales Heatmap: Region × Month')
plt.tight_layout()

Pairplot — all combinations at once

# Great for initial exploration of a multi-column dataset
g = sns.pairplot(
    df[['sales','commission','hours_worked','deals_closed']],
    diag_kind='kde',           # diagonal: KDE instead of histogram
    plot_kws={'alpha': 0.5},
    hue='region'               # colour by category
)
g.fig.suptitle('Pairwise Relationships', y=1.02)
plt.savefig('pairplot.png', dpi=120, bbox_inches='tight')
Chapter 8
Real-World Chart Workflows

Workflow 1 — Monthly Sales Dashboard (saved to PNG)

import pandas as pd
import matplotlib.pyplot as plt
import matplotlib.gridspec as gridspec
from pathlib import Path

df = pd.read_csv('sales.csv', parse_dates=['date'])
df['month'] = df['date'].dt.to_period('M').astype(str)

monthly   = df.groupby('month')['sales'].sum()
by_region = df.groupby('region')['sales'].sum().sort_values()
top_reps  = df.groupby('rep_name')['sales'].sum().nlargest(5)

fig = plt.figure(figsize=(14, 8), facecolor='#f8f9fa')
gs  = gridspec.GridSpec(2, 3, figure=fig, hspace=0.45, wspace=0.35)

ax1 = fig.add_subplot(gs[0, :2])   # wide trend
ax2 = fig.add_subplot(gs[0, 2])    # region bars
ax3 = fig.add_subplot(gs[1, :])    # full-width top reps

# Trend
ax1.plot(monthly.index, monthly.values, color='#2980b9', linewidth=2.5, marker='o')
ax1.fill_between(range(len(monthly)), monthly.values, alpha=0.15, color='#2980b9')
ax1.set_title('Monthly Revenue', fontweight='bold')
ax1.set_ylabel('Sales ($)')
ax1.tick_params(axis='x', rotation=45)

# Region
ax2.barh(by_region.index, by_region.values, color='#5dade2')
ax2.set_title('By Region', fontweight='bold')

# Top reps
bars = ax3.bar(top_reps.index, top_reps.values, color='#1abc9c', edgecolor='white')
ax3.bar_label(bars, fmt='${:,.0f}', padding=4, fontsize=9)
ax3.set_title('Top 5 Sales Reps', fontweight='bold')
ax3.tick_params(axis='x', rotation=20)

for ax in [ax1, ax2, ax3]:
    ax.spines[['top','right']].set_visible(False)
    ax.grid(True, axis='y', alpha=0.3)

fig.suptitle('Sales Dashboard', fontsize=16, fontweight='bold', y=1.01)
plt.savefig('dashboard.png', dpi=150, bbox_inches='tight', facecolor=fig.get_facecolor())
print("Saved dashboard.png")

Workflow 2 — Automated Chart per Region

from pathlib import Path

Path('charts').mkdir(exist_ok=True)

for region, group in df.groupby('region'):
    monthly = group.groupby('month')['sales'].sum()

    fig, ax = plt.subplots(figsize=(8, 4))
    ax.plot(monthly.index, monthly.values, marker='o', linewidth=2, color='#3498db')
    ax.fill_between(range(len(monthly)), monthly.values, alpha=0.2)
    ax.set_title(f'{region} Region — Monthly Sales', fontweight='bold')
    ax.set_ylabel('Sales ($)')
    ax.tick_params(axis='x', rotation=45)
    ax.spines[['top','right']].set_visible(False)
    plt.tight_layout()
    plt.savefig(f'charts/{region.lower()}_sales.png', dpi=150, bbox_inches='tight')
    plt.close()   # ← important: free memory when generating many charts
    print(f"Saved chart for {region}")
Always plt.close() in loops If you generate charts in a loop without closing them, Matplotlib accumulates open figures in memory. Use plt.close() or plt.close('all') after saving.
Matplotlib vs Seaborn — when to use which
  • Matplotlib — full control over every pixel; best for dashboards, custom styling, complex layouts
  • Seaborn — statistical plots in one line; best for EDA, distributions, correlations when the DataFrame structure matches directly
  • They compose: use sns.xxx(ax=ax) to draw Seaborn plots onto a Matplotlib Axes object
