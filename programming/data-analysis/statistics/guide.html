Chapter 1
Descriptive Statistics

Descriptive statistics summarise a dataset. Before any modelling or testing, you should always compute these to understand what you have.

Measures of Central Tendency

Mean = (x₁ + x₂ + ... + xₙ) / n
arithmetic average — sensitive to outliers
Median = middle value when sorted
50th percentile — robust to outliers
Mode = most frequently occurring value
useful for categorical data
import numpy as np
import pandas as pd
from scipy import stats

data = [12400, 15800, 9700, 18300, 14200, 9700, 22000, 11500]

np.mean(data)     # 14200.0
np.median(data)   # 13300.0
stats.mode(data)  # ModeResult(mode=9700, count=2)

# With Pandas
s = pd.Series(data)
s.mean()
s.median()
s.mode()[0]

Measures of Spread

Variance σ² = Σ(xᵢ − x̄)² / n
average squared deviation from mean
Standard Deviation σ = √Variance
same units as data — most interpretable spread measure
IQR = Q3 − Q1
interquartile range — spread of middle 50% — robust to outliers
np.std(data, ddof=0)   # population std (divide by n)
np.std(data, ddof=1)   # sample std (divide by n-1) ← use this for samples

s.std()                # same as ddof=1
s.var()                # variance
s.quantile(0.25)       # Q1
s.quantile(0.75)       # Q3
s.quantile(0.75) - s.quantile(0.25)  # IQR

s.describe()   # all of the above in one call

Shape: Skewness & Kurtosis

s.skew()     # 0 = symmetric, >0 = right-skewed, <0 = left-skewed
s.kurtosis() # 0 = normal, >0 = heavier tails, <0 = lighter tails
Mean vs Median — which to use? For symmetric data, they are similar. For skewed data (e.g. income, house prices, order values), the median is a better "typical" value because it is not pulled by extreme outliers. Always check a histogram before deciding.
Chapter 2
Distributions

A distribution describes how values are spread across a dataset. Understanding the shape of your data is essential before choosing models or tests.

The Normal Distribution

The most important distribution in statistics. Many real-world measurements (heights, measurement errors, aggregated sums) follow a bell curve.

The 68-95-99.7 rule: in a normal distribution, 68% of data falls within 1 standard deviation of the mean, 95% within 2, and 99.7% within 3.

from scipy.stats import norm
import numpy as np

mu, sigma = 50, 10   # mean=50, std=10

# Probability that X ≤ 65
norm.cdf(65, loc=mu, scale=sigma)   # 0.9332 → 93.3%

# Value at which 90% of data falls below (90th percentile)
norm.ppf(0.90, loc=mu, scale=sigma) # 62.8

# Generate 1000 samples
samples = norm.rvs(loc=mu, scale=sigma, size=1000, random_state=42)

Common Distribution Shapes

from scipy.stats import norm, expon, lognorm
import matplotlib.pyplot as plt, numpy as np

x = np.linspace(0, 5, 300)
fig, axes = plt.subplots(1, 3, figsize=(12, 3))

axes[0].plot(np.linspace(-4,4,300), norm.pdf(np.linspace(-4,4,300)))
axes[0].set_title('Normal (symmetric)')
axes[0].fill_between(np.linspace(-4,4,300), norm.pdf(np.linspace(-4,4,300)), alpha=0.3)

axes[1].plot(x, expon.pdf(x))
axes[1].set_title('Exponential (right-skewed)')
axes[1].fill_between(x, expon.pdf(x), alpha=0.3)

axes[2].plot(x, lognorm.pdf(x, s=0.6))
axes[2].set_title('Log-normal (right-skewed)')
axes[2].fill_between(x, lognorm.pdf(x, s=0.6), alpha=0.3)

plt.tight_layout()
plt.savefig('distributions.png', dpi=120)

Central Limit Theorem

No matter what the original distribution looks like, the distribution of sample means will approach a normal distribution as the sample size grows. This is why so many statistical tests assume normality.

import numpy as np
import matplotlib.pyplot as plt

# Original data: skewed (exponential)
population = np.random.exponential(scale=2, size=100000)

# Sample means of size n=30
means = [np.mean(np.random.choice(population, size=30)) for _ in range(5000)]

fig, axes = plt.subplots(1, 2, figsize=(10, 4))
axes[0].hist(population, bins=80, color='#e74c3c', alpha=0.7)
axes[0].set_title('Original population (skewed)')
axes[1].hist(means, bins=60, color='#2980b9', alpha=0.7)
axes[1].set_title('Sampling distribution of means\n(n=30) — approaches normal')
plt.tight_layout()
Chapter 3
Correlation

Correlation measures how strongly two variables move together.

r = Σ[(xᵢ − x̄)(yᵢ − ȳ)] / [n · σₓ · σᵧ]
Pearson correlation coefficient — ranges from −1 to +1
import pandas as pd, numpy as np

df = pd.read_csv('sales.csv')

# Correlation matrix — all numeric columns
corr = df.corr(numeric_only=True)
print(corr)

# Single pair
r = df['hours_worked'].corr(df['sales'])
print(f"r = {r:.3f}")

# Spearman (rank-based, robust to outliers and non-linear)
from scipy.stats import spearmanr, pearsonr

r_p, p_p = pearsonr(df['hours_worked'], df['sales'])
r_s, p_s = spearmanr(df['hours_worked'], df['sales'])

print(f"Pearson  r={r_p:.3f}  p={p_p:.4f}")
print(f"Spearman r={r_s:.3f}  p={p_s:.4f}")
Correlation ≠ causation A high correlation means two variables move together, not that one causes the other. Ice cream sales correlate with drowning deaths (both rise in summer). Always consider confounding variables.

Interpreting r

  • |r| ≥ 0.7 — strong correlation
  • 0.4 ≤ |r| < 0.7 — moderate correlation
  • 0.1 ≤ |r| < 0.4 — weak correlation
  • |r| < 0.1 — negligible (near zero)
  • Sign: positive = both increase together; negative = one increases as other decreases
Chapter 4
Probability Basics

Statistics is built on probability. You don't need to be a mathematician — but understanding these ideas is essential for interpreting p-values and confidence intervals.

Key concepts

  • Event — an outcome or set of outcomes (e.g. "sale > $10k")
  • Probability — a number between 0 and 1 representing likelihood (0 = impossible, 1 = certain)
  • P(A and B) = P(A) × P(B) if independent
  • P(A or B) = P(A) + P(B) − P(A and B)
  • Conditional probability P(A|B) — probability of A given B has occurred
import numpy as np

# Simulate: what % of days does sales exceed $15k?
np.random.seed(42)
daily_sales = np.random.normal(loc=13000, scale=3000, size=10000)

p_exceed = np.mean(daily_sales > 15000)
print(f"P(sales > 15k) = {p_exceed:.3f}  ({p_exceed*100:.1f}%)")

# Expected value
outcomes  = np.array([5000, 10000, 20000, 40000])
probs     = np.array([0.3,  0.4,   0.2,   0.1])
ev = np.sum(outcomes * probs)
print(f"Expected daily sales = ${ev:,.0f}")
Chapter 5
Confidence Intervals

A confidence interval gives a range of plausible values for a population parameter, based on sample data. A 95% CI means: if you repeated the experiment 100 times, 95 of those intervals would contain the true value.

CI = x̄ ± z × (σ / √n)
for a 95% CI: z = 1.96 · x̄ = sample mean · σ = std · n = sample size
import numpy as np
from scipy import stats

sample = np.array([12400, 15800, 9700, 18300, 14200, 13800, 16500, 11900])
n    = len(sample)
mean = np.mean(sample)
se   = stats.sem(sample)   # standard error = std / sqrt(n)

# 95% confidence interval
ci_95 = stats.t.interval(confidence=0.95, df=n-1, loc=mean, scale=se)
print(f"Mean: ${mean:,.0f}")
print(f"95% CI: (${ci_95[0]:,.0f}, ${ci_95[1]:,.0f})")

# Interpret: we're 95% confident the true average sale is in this range
Wider CIs = more uncertainty Small sample size, high variance, or a higher confidence level (99% vs 95%) all widen the interval. More data narrows it.
Chapter 6
Hypothesis Testing

Hypothesis testing is the framework for making data-driven decisions. You start with an assumption (null hypothesis), collect data, and assess whether there is enough evidence to reject it.

The framework

  1. H₀ (Null hypothesis) — "There is no effect / no difference" (default assumption)
  2. H₁ (Alternative hypothesis) — "There IS an effect / difference"
  3. Collect data and compute a test statistic
  4. Compute the p-value — probability of seeing this result (or more extreme) if H₀ is true
  5. If p-value < significance level (usually 0.05), reject H₀
What the p-value really means p = 0.03 means: "If there were truly no difference, there is only a 3% chance of observing data as extreme as this." It does NOT mean "there is a 3% chance the null hypothesis is true."

Type I & Type II Errors

H₀ is True H₀ is False
Reject H₀ Type I Error (false positive) — α Correct ✓
Fail to reject H₀ Correct ✓ Type II Error (false negative) — β

t-Test — compare two means

from scipy import stats
import numpy as np

# One-sample t-test: is the mean different from a target?
sample = [12400, 15800, 9700, 18300, 14200, 13800]
t_stat, p_val = stats.ttest_1samp(sample, popmean=13000)
print(f"t={t_stat:.3f}, p={p_val:.4f}")
print("Reject H0" if p_val < 0.05 else "Fail to reject H0")

# Two-sample t-test: are two groups different?
group_a = [12400, 15800, 13100, 14700, 11900]
group_b = [18300, 21400, 19800, 17600, 22100]
t_stat, p_val = stats.ttest_ind(group_a, group_b, equal_var=True)
print(f"t={t_stat:.3f}, p={p_val:.4f}")

# Paired t-test: before vs after same subjects
before = [12400, 9800, 15600, 11200, 13700]
after  = [14800, 12100, 17200, 13400, 15300]
t_stat, p_val = stats.ttest_rel(before, after)
print(f"Paired t: t={t_stat:.3f}, p={p_val:.4f}")

Chi-Square Test — categorical variables

from scipy.stats import chi2_contingency

# Is region independent of high/low sales?
observed = np.array([
    [12, 8],   # North: [high, low]
    [18, 5],   # South
    [9,  11],  # East
    [6,  14],  # West
])
chi2, p_val, dof, expected = chi2_contingency(observed)
print(f"Chi2={chi2:.3f}, p={p_val:.4f}, dof={dof}")
print("Significant association" if p_val < 0.05 else "No significant association")

ANOVA — compare more than two groups

from scipy.stats import f_oneway

north = [12400, 9700, 14200, 11500, 13800]
south = [15800, 18300, 14200, 19100, 16700]
east  = [11200, 13400, 10800, 14500, 12300]

f_stat, p_val = f_oneway(north, south, east)
print(f"F={f_stat:.3f}, p={p_val:.4f}")
# p<0.05: at least one group mean is significantly different

A/B Test walkthrough

import numpy as np
from scipy import stats

np.random.seed(42)
# Control: old checkout button (conversion rate 12%)
# Treatment: new button (conversion rate 14.5%)
n = 1000
control   = np.random.binomial(1, 0.12, n)
treatment = np.random.binomial(1, 0.145, n)

print(f"Control rate:   {control.mean()*100:.1f}%")
print(f"Treatment rate: {treatment.mean()*100:.1f}%")

# t-test on proportions
t_stat, p_val = stats.ttest_ind(control, treatment)
print(f"p-value: {p_val:.4f}")

if p_val < 0.05:
    lift = (treatment.mean() - control.mean()) / control.mean() * 100
    print(f"Significant! Lift = {lift:.1f}% — deploy the new button.")
else:
    print("Not significant — gather more data.")
Practical significance vs statistical significance A result can be statistically significant (p < 0.05) but have a tiny real-world effect — especially with large datasets. Always report effect size alongside p-values. Cohen's d for means, odds ratio for proportions.
